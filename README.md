# App Restaurante de Comida

Un restaurante de comida local cuenta con dos paneles, el panel de Cliente y el panel de Administrador:
### Panel de Cliente
El panel de cliente cuenta con las siguientes funcionalidades:
- Inicio de sesion
- Registro de cuenta con envio de correo
- Restablecer contraseña 
- Crear una orden en base a los platillos ofertados
- Eliminar una orden 
- Editar una orden
- Visualizar una orden
- Visualizar los platillos pertenecientes a una orden
- Editar cuenta
- Eliminar Cuenta
- Cerrar Sesion

### Panel de Administrador
El panel de administrador cuenta con las siguientes funcionalidades:
- Inicio de sesion
- Registro de cuenta 
- Restablecer contraseña 
- Crear una platillo que sera ofertado al cliente 
- Eliminar un platillo 
- Editar un platillo
- Visualizar platillos
- Visualizar todas las ordenes creadas
- Editar cuenta
- Eliminar Cuenta
- Cerrar Sesion
